# بیزانە — ڤێرژنی ئەندرۆید

ئەمە پۆرتێکی تەواوی ئەپی iOS (بیزانە) بۆ ئەندرۆیدە، بە Kotlin + Jetpack Compose نووسراوە.
هەردوو ئەپەکە (iOS و ئەندرۆید) **هەمان پڕۆژەی Firebase** بەکاردەهێنن (هەمان `FirebaseConfig`),
بۆیە ئەگەر لەسەر ئایفۆن گروپێک دروستکرد، دەتوانیت بەهەمان کۆدی گروپ لەسەر ئەندرۆیدیش بچیتە ناوی
و هەردووکیان یەک داتا نیشان بدەن.

## چۆنیەتی کردنەوەی پڕۆژەکە

1. Android Studio (نوێترین وەشان) دابمەزرێنە.
2. **File → Open** بکە و فۆڵدەری `BizaneAndroid` هەڵبژێرە (ئەم فۆڵدەرەی ئێستا تێیدایت).
3. Android Studio خۆی Gradle sync دەکات. ئەم پڕۆژەیە `gradlew`ی خۆی نییە (چونکە
   gradle-wrapper.jar فایلێکی binary ـە کە نەمتوانی دروستی بکەم لێرەوە) — Android Studio
   بەشێوەیەکی خۆکار پێشنیاری دروستکردنی wrapper دەکات یان بە Gradle ـی داخڵیی خۆی sync دەکات.
   ئەگەر داوات لێکرد "Create Gradle Wrapper" یان shتی هاوشێوە، قبوڵی بکە.
4. دوای sync بوونی سەرکەوتوو، **Run ▶** بکە لەسەر ئامێرێک یان ئیمولەیتەرێک (Android 8.0 / API 26+).

## تایبەتمەندییەکان (هەمووی وەکو ئایفۆن)

- زیادکردن/دەستکاریکردن/سڕینەوەی خواردن، بە وێنە و OCR (ML Kit، لە جیاتی Vision‌ی ئەپڵ)
- گروپی هاوبەش (دروستکردن/چوونەناو/جیابوونەوە)، بە ئەدمین و ئەندام
- تایبەتمەندی: ئەندام تەنیا ئایتمی خۆی دەبینێت، ئەدمین هەمووی دەبینێت بەڵام تەنیا دەتوانێت ئایتمی خۆی بگۆڕێت
- کردنەوەی گشتیی دەسەڵاتی سڕینەوە بە یوزەر/پاس (`em25` / `X@x1997`) — هەمان یاداشتی ئاسایشی پێشوو دەبەخشرێت
- Undo دوای سڕینەوە (Snackbar)
- گەڕان و ڕیزکردن (نزیکترین/دوورترین بەسەرچوون، ناو)
- کاتی دواین نوێکردنەوە + کاش‌کردنی لۆکاڵی بۆ کاتی بێ‌ئینتەرنێت
- ڕێکخستنەکان (ئاگادارکردنەوە، ئامار، دەربارە، پەیوەندی)

## ئەوانەی هێشتا نەکراون یان جیاوازن

- **ئاگادارکردنەوەی ڕاستەقینە (push notification)**: لە ئایفۆنیش هێشتا کارا نەکرابوو، لێرەش نەکراوە.
- **App icon**: ئایکۆنێکی سادەی vector دانراوە (نەک دیزاینێکی تەواو)، دەتوانیت بیگۆڕیت لە
  `app/src/main/res/mipmap-anydpi-v26/` و `drawable/ic_launcher_*.xml`.
- تایبەتمەندی "دەسەڵاتی سڕینەوە بۆ هەمووان" و فلتەرکردنی نهێنی تەنیا لای **کلاینت**ـە
  (هەروەک لە ئایفۆنیش)، نەک Firestore Security Rules. هەر کەسێکی شارەزا کە داواکارییە
  تۆڕییەکان ببینێت دەتوانێت داتای ڕاو ببینێت.

## تێبینی گرنگ

ئەم کۆدە بە وردی و بەبێ توانای build/compile کردن لێرەدا نووسراوە (ئەم ژینگەیە Android SDK
یان Gradle ـی تێدا نییە). زۆرترین بەشەکانی وەرگێڕان لە Swift/UIKit بۆ Kotlin/Compose بە وردی
چاودێری کراون، بەڵام لەوانەیە هەندێک هەڵەی بچووکی build (بۆ نموونە ناوی پارامیتەرێکی API‌یەکی
Compose کە گۆڕاوە لە وەشانێکەوە بۆ وەشانێک) دەربکەوێت کاتێک یەکەم جار sync/build دەکەیت.
ئەگەر هەڵەیەکت بۆ دەرکەوت، دەقی هەڵەکە (error message) بۆم بنێرەوە، یارمەتیت دەدەم چاکی بکەمەوە.
